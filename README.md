# HireSmart AI

HireSmart AI is an AI-powered Recruitment & Career Management Platform built using Django.

## Features

- User Authentication
- Resume Upload
- AI Resume Analysis
- ATS Score
- Job Portal
- Job Applications
- Recruiter Dashboard
- Admin Dashboard
- REST APIs
- AWS Deployment

## Technology Stack

- Django
- Django REST Framework
- PostgreSQL
- Bootstrap
- JavaScript
- Gemini API
- Docker
- Gunicorn
- Nginx
- AWS EC2
